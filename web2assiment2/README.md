# Backend Weather Application


## Setup
1. Clone repository
2. npm install
3. Create .env file and add OpenWeather API key
4. npm start


## APIs Used
- OpenWeather API (Weather data)
- REST Countries API (Country information)


## Design Decisions
- All API requests handled on the server
- Clean separation between backend and frontend
- Responsive and minimal UI
